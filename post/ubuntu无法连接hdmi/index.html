ubuntu无法连接hdmi

2024-04-10 约 290 字 预计阅读 1 分钟

文章目录

安装了几个第三方包重启系统后突然发现无法识别hdmi外接屏幕了

执行 sudo lshw -C video 显示如下信息,即没有分配显卡驱动

*-display UNCLAIMED description: VGA compatible controller

同时 nvidia-smi 显示如下信息

NVIDIA-SMI has failed because it couldn’t communicate with the NVIDIA driver. Make sure that the latest NVIDIA driver is installed and running.

可以确认是显卡没有配置好的问题

回想起来,问题的应该是安装了新的包后需要执行 perform mok managemnet 相关步骤,当时忽略了导致显卡没有正常启动

重启后出现蓝屏的perform mok managemnet 按照以下步骤进行：

选择enroll mok（perform mok mangement界面）

选择continue（enroll mok界面）

选择yes（enroll the key界面）

输入在安装时设定的密码

选择Reboot（perform mok manageent界面）
